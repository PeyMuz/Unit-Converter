# Unit Converter

A modern Python unit converter app with a neon "techy" theme. Instantly convert Length, Mass, and Volume units with a stylish animated interface.  
Built using Tkinter for a smooth and visually appealing desktop experience.

## Features
- Supports Length, Mass, and Volume conversions
- Neon dark mode UI with animated tech lines
- Easy-to-use dropdowns and input fields
- Responsive and visually appealing design

## How to Run

1. Make sure you have Python 3 installed.
2. Download or clone this repository.
3. Run the app:
   ```
   python "Unit Converter.py"
   ```

## Example Usage

- Select a unit type (e.g., Length)
- Choose the units to convert from and to (e.g., Kilometer to Meter)
- Enter a value (e.g., 12)
- Click **Answer** to see the result
